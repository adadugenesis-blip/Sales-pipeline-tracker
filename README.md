
# Project: Sales Pipeline Tracker

## 1. Project Introduction

This project helps students understand how businesses track potential customers from first contact to closed sales.

Students will use Excel to organize sales opportunities, monitor deal stages, and calculate expected revenue.

By completing this project, students will learn how Excel supports sales teams in decision-making, forecasting revenue, and identifying which deals need attention.

---

## 2. Skills You Will Obtain After Completing This Project

After working on this project, students will be able to:

1. Organize and clean sales data in Excel  
2. Track sales opportunities using pipeline stages  
3. Use logical and conditional Excel functions  
4. Calculate expected and closed revenue  
5. Analyze sales performance by stage  
6. Create summary tables for sales tracking  
7. Visualize pipeline data using charts  

---

## 3. Project Instructions (Tasks)

1. Calculate the **Expected Revenue** for each deal.  
2. Calculate the **Total Expected Revenue** for all open deals.  
3. Use `SUMIF` to calculate total deal value for each pipeline stage.  
4. Count how many deals are in each stage.  
5. Calculate total revenue from **Closed Won** deals only.  
6. Create a **sales summary section** showing:
   - Total deals  
   - Total expected revenue  
   - Closed won revenue  

---
